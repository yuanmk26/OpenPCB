# OpenPCB Project Management Docs

This directory stores project management documents separately from architecture and change logs.

## Documents

- `docs/project/TODO.md`: source of milestones, status, and acceptance criteria.
- `docs/project/CI_CD.md`: CI/CD goals, triggers, quality gates, release strategy, and failure handling.

## Maintenance Rules

- Update `docs/project/TODO.md` whenever milestone scope, priority, status, or acceptance criteria changes.
- Update `docs/project/CI_CD.md` whenever CI/CD process or release policy changes.
- Add a same-day change record under `docs/changes/<YYYY-MM-DD>/` for every update.
