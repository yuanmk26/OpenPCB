## 变更前问题

原理图预览默认开启调试图层，且正式图元大量使用非缩放线宽，叠加过密的页面网格后，整页适配时会出现器件、线条和网格都显得偏粗偏大的问题。

## 变更内容

关闭原理图预览默认调试模式；增大页面网格步距并减小网格线宽与透明度；移除正式原理图图元上的非缩放线宽，让线宽随页面缩放一起变化，同时略微收细导线与页框描边。

## 影响范围

影响 GUI 原理图预览的默认视觉表现，包括默认图层、网格密度和正式图元线宽缩放行为，不影响几何数据结构和交互逻辑。

## 验证结果

执行 `npm run build`（目录：`src/gui`），构建通过。

## 下一步建议

建议后续把页面单位、网格单位、符号库尺寸和默认线宽整理成统一常量，避免继续在不同文件里用经验值分别调节。
