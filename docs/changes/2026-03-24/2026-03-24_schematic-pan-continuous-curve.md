## 变更前问题

原理图预览的拖动阻尼依赖分段阈值，虽然可以逐步调慢拖动速度，但不同缩放区间之间的手感变化不够平滑，调参成本也较高。

## 变更内容

将原理图预览的拖动速度改为固定可调参数驱动的连续函数，并让拖动阻尼与当前缩放比例严格绑定，替代原先的分段经验值。

## 影响范围

影响 GUI 原理图预览拖动交互的速度曲线和调参方式，不改变页面内容、缩放逻辑和渲染数据结构。

## 验证结果

执行 `npm run build`（目录：`src/gui`），构建通过。

## 下一步建议

如果后续还要继续调优，建议把这组拖动参数暴露为统一的 viewport 配置常量，并结合滚轮缩放一并校准。
