## 变更前问题

原理图预览拖动画布时仍依赖屏幕绝对坐标差值，鼠标会持续在窗口内移动，长距离拖动时容易碰到屏幕边界，影响拖动连续性和交互稳定性。

## 变更内容

为原理图预览拖动增加 Pointer Lock 支持：开始拖动时优先请求锁定鼠标，拖动过程中改用相对位移 `movementX/movementY` 计算平移；同时保留原有坐标差值逻辑作为不支持 Pointer Lock 的兜底路径，并在锁定释放时自动结束拖动状态。

## 影响范围

影响 GUI 原理图预览的拖动交互，仅调整拖动时鼠标输入处理方式，不修改当前缩放行为和拖动速度参数。

## 验证结果

执行 `npm run build`（目录：`src/gui`），构建通过。

## 下一步建议

建议后续为 Pointer Lock 状态补一个轻量提示，例如状态栏或光标样式变化，避免用户不知道可通过 `Esc` 退出锁定。
