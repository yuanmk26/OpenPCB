## 变更前问题

原理图预览在拖动画布时，浏览器仍会对 SVG 内的文字和部分元素执行文本选择，导致器件标注或文字出现高亮选中，影响拖动交互体验。

## 变更内容

在原理图预览的指针拖动事件中补充 `preventDefault()`，并为画布容器与 SVG 画布增加 `user-select: none`，阻止拖动过程中触发文本选择高亮。

## 影响范围

影响 GUI 原理图预览拖动交互，仅调整拖动时的浏览器默认选择行为，不修改缩放逻辑、拖动速度参数和渲染数据。

## 验证结果

执行 `npm run build`（目录：`src/gui`），构建通过。

## 下一步建议

建议后续补一层拖动状态样式，例如拖动时临时禁用页面级选择和光标反馈，以保持不同输入设备下的交互一致性。
