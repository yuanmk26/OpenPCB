## 变更前问题

原理图预览默认按内容包围盒执行 fit，页面内容较少时会被过度放大，导致预览初始视图离得太近，不利于在窗口内整体查看页面比例与布局。

## 变更内容

调整 `fitViewportToContent` 的缩放计算，使内容适配比例不会超过整页适配比例；同时将原理图预览默认视图和重置行为切换为整页适配。

## 影响范围

影响 GUI 原理图预览的初始缩放、重置缩放以及内容适配时的最大放大比例，不影响原理图数据结构和绘制元素本身。

## 验证结果

执行 `npm run build`（目录：`src/gui`），构建通过。

## 下一步建议

如果后续接入真实原理图数据，建议补充 viewport 计算单测，并区分“整页预览”和“内容聚焦”两类交互预期。
