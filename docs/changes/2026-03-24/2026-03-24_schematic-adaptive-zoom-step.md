## 变更前问题

原理图预览在整页缩放比例很低时，工具栏 `Zoom In` 仍按固定小步长放大，导致点击后肉眼几乎看不出变化，交互反馈不明显。

## 变更内容

将原理图预览的工具栏缩放改为自适应步长：在低缩放比例下使用更大的放大/缩小倍率，进入较高缩放级别后再切回较细的步进。

## 影响范围

影响 GUI 原理图预览工具栏缩放的交互反馈，不改变页面内容、viewport 约束和渲染数据结构。

## 验证结果

执行 `npm run build`（目录：`src/gui`），构建通过。

## 下一步建议

建议后续把按钮缩放、滚轮缩放和双击缩放统一为同一套自适应缩放策略，并提取为公共 viewport 常量。
